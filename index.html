CSS is awesome

Why CSS is awesome

A little CSS can easily transform a single HTML element into a fullscreen animation without loading additional resources or require extra markup.

That's right. Your lean semantic HTML doesn't need to be compromised. You don't need extra trips to the server. And don't worry about high resolution displays. CSS got you covered there too.

How CSS is awesome

Each HTML element is like a canvas to CSS. You can paint it with multiple background gradients. You can add box-shadows for a final touch. Heck, you can even animate it.

And if you ever find yourself running out of canvas, CSS is kind enough to give you two extra pseudo elements for each element. That's three canvases for the price of one.

About this page

This 10 kilobyte page is fully self contained. It's build on the principles of progressive enhancement. HTML-only readers get the core message. CSS capable browsers renders a demo too.

If JavaScript is enabled, the demo becomes interactive through injected controls.
